# 🎬 Vyoma AI Demo - Quick Start Guide

## ⚡ 5-Minute Setup

### 1. Start Services (2 minutes)
```bash
# Terminal 1: Backend
cd backend
python app.py

# Terminal 2: Frontend  
cd frontend
python -m http.server 8000

# Verify: Open http://localhost:8000/index.html in browser
```

### 2. Prepare OBS (2 minutes)
```
1. Download: https://obsproject.com
2. Open OBS Studio
3. Add Display Capture source → Select browser window
4. Add Audio Input → Select microphone
5. Set output folder for recording
```

### 3. Start Recording (1 minute)
```
1. Click "Start Recording" in OBS
2. Follow script from DEMO_VIDEO_SCRIPT.md
3. Record all 5 scenes (60 seconds total)
4. Click "Stop Recording"
```
